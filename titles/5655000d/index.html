compat_comments: Overall, the entire game is very playable, up through the final cutscene.

There were dips in performance, but not too often and typically not major enough to severely impact gameplay.

The more problematic glitches observed include:
-Leaving xemu on in the background (est. 10-30 minutes) with this game running can lead to the game freezing, necessitating a restart via the xemu menus
-UI flickering, sometimes with UI elements being replaced with erroneous assets e.g. different parts of the targeting reticle replaced by machine gun icons
-Characters getting stuck along walls - unsticking without reloading/restarting can be hard
-Restarting a level can make the grenade launcher unusable - can fix by dropping exactly one launcher
-Game freezing/corrupt save files when saving in the Dome
-Bringing up a menu immediately as you are given control for the very last time during the last level may result in xemu becoming unresponsive, necessitating that the process be killed
